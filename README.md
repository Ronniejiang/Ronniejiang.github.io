# Chengjie Jiang Academic Homepage

This repository contains the source for Chengjie Jiang's academic homepage, built with the Academic Pages Jekyll template and intended for deployment at:

<https://ronniejiang.github.io>

## Local Preview

Use Ruby 3 or newer. The macOS system Ruby 2.6 is too old for the current GitHub Pages dependency stack.

```bash
bundle install
bundle exec jekyll serve
```

Then open <http://127.0.0.1:4000>.

The site includes profile information, publications, internship information, awards, and a downloadable CV.
